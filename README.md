# node基础
## 常用命令
### npm install -g supervisor   ---- supervisor app.js  ---同步修改文件
### npm init --yes  ---生成package.json文件
### npm install/npm uninstall
### npm list  ---列出安装包
### npm info jquery  ---查看所有版本
### npm install jquery@1.8.2   ---安装指定版本

## "ejs": "^2.3.4"
### ^表示第一位版本号不变，后面两位取最新的
### ~表示前两位不变，最后一个取最新
### *表示全部取最新
